# wasmGL
Basic OpenGL application served with WebAssembly

## Generating Makefiles using Cmake
### wasm
make cmake-wasm
### native
make cmake-native
### native windows from wsl
make cmake-windows

## Building
### wasm
make wasm
### native
make native
### native windows from wsl
make windows

## Requirements
- glfw3.3
- glad
- openGL
